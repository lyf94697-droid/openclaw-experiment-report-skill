# 进程调度算法模拟实验

请生成一份中文大学实验报告正文，课程为“操作系统”，实验名称为“进程调度算法模拟”。

## 实验事实

- 实验比较 FCFS、SJF 和时间片轮转三种调度算法
- 输入进程包含到达时间、服务时间和优先级字段
- 程序输出每个进程的开始时间、完成时间、周转时间和带权周转时间
- 实验重点是比较不同调度策略对平均等待时间和平均周转时间的影响

## 写作要求

- 按“实验目的、实验环境、实验原理或任务要求、实验步骤、实验结果、问题分析、实验总结”组织
- 解释算法原理时要简洁，不要堆砌教材内容
- 结果分析要围绕等待时间、周转时间和公平性
- 缺少真实截图时，不要编造截图内容
